Usage Examples:
---------------

1. Using Facade (Simplest):
   Campaign::send('Welcome Campaign', $phone, $variables);

2. In Filament Resource:
   protected function afterCreate(): void {
       Campaign::send('Order Confirmation', 
           $this->record->phone, $variables);
   }

3. Using Model Trait:
   class Order extends Model {
       use SendsCampaignMessages;
   }
   $order->sendCampaign('Order Confirmation', $phone, $vars);

4. Queue Job:
   SendCampaignMessageJob::dispatch($campaign, $phone, $vars);


Installation:
-------------

1. Seed campaigns:
   php artisan db:seed --class=CommonCampaignsSeeder

2. (Optional) Add facade alias in config/app.php:
   'Campaign' => App\Facades\Campaign::class,

3. (Optional) Start queue worker:
   php artisan queue:work --queue=whatsapp,whatsapp-bulk,default
